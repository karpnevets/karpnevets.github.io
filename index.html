User Guide


1. 기본 설정 방법
팀 이름 설정은, 팀 이름 입력란을 클릭해 작성 가능합니다.

2. 점수 조작 방법
점수는 단축키를 통해 조작 할 수 있습니다. 왼쪽 팀의 점수는 "Q", "A", 오른쪽 팀의 점수는 "Y","H"를 통해 각각 1점 식 추가하고 뺄 수 있습니다.

3. 타이머 조작 방법
타이머 시간 설정은 우측 하단의 "Time Setting"을 통해 초 단위로 입력하면 됩니다. 타이머를 시작할 때는 "S",일시정지 할 때는 "P"를 사용 할 수 있습니다.

4. 추가 기능
하단의 "Reset" 버튼을 통해서는 양 팀의 점수, 이름, 시간을 초기화 할 수 있고, "Switch" 버튼을 통해 양 팀의 점수와 팀명의 위치를 바꿀 수 있습니다.
